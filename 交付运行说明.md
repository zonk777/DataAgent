# 数据智能体服务系统 - 交付运行说明

## 1. 项目说明

本压缩包为“数据智能体服务系统”完整项目源码与文档交付包，包含前端、后端、项目文档、测试数据和本地存储数据。

系统主要功能：

- 管理员账号密码登录
- CSV 数据集上传与字段识别
- 自然语言智能数据分析
- 多轮会话与历史对话恢复
- 知识库片段管理与知识问答
- 图表、查询表格、SQL 和分析结论展示
- 系统配置状态查看

## 2. 目录结构

```text
数据智能体服务系统
├─ backend              后端 FastAPI 服务
├─ frontend             前端 Vue3 页面
├─ docs                 项目文档、需求说明、概要设计文档
├─ scripts              启动脚本与文档生成脚本
├─ test-data            测试 CSV 数据
└─ README.md            项目基础说明
```

## 3. 环境要求

建议对方电脑安装：

- Python 3.11 或以上
- Node.js 18 或以上
- Edge/Chrome 浏览器

## 4. 后端启动

打开命令行，进入项目后端目录：

```bat
cd backend
python -m venv .venv
.venv\Scripts\python.exe -m pip install -r requirements.txt
.venv\Scripts\python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

后端启动成功后，浏览器访问：

```text
http://127.0.0.1:8000/docs
```

如果能看到接口文档，说明后端正常。

## 5. 前端启动

另开一个命令行窗口，进入前端目录：

```bat
cd frontend
npm install
npm run dev
```

前端启动后，浏览器访问终端显示的地址，一般是：

```text
http://127.0.0.1:5173
```

## 6. 登录说明

打开前端页面后会先进入登录页。

初始管理员账号和密码请以项目实际配置为准。

## 7. API Key 说明

本交付包保留了 backend/.env 配置文件。模型 API Key、LLM 地址、Embedding 地址等配置均由后端读取，前端页面不会直接显示密钥。

如果换电脑运行失败，优先检查 backend/.env 中的模型服务配置是否仍然可用。

## 8. 常见问题

### 8.1 前端打不开 localhost:5173

通常是前端没有启动。请确认已经在 frontend 目录执行：

```bat
npm run dev
```

### 8.2 后端 uvicorn 不是内部或外部命令

请使用虚拟环境里的 Python 启动：

```bat
.venv\Scripts\python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

### 8.3 模型显示未配置

检查 backend/.env 中是否配置：

```text
LLM_API_KEY=
LLM_BASE_URL=
LLM_MODEL=
EMBEDDING_API_KEY=
EMBEDDING_BASE_URL=
EMBEDDING_MODEL=
```

配置完成后需要重新启动后端。

### 8.4 知识库问答不可用

知识库问答依赖 Embedding 配置和 Qdrant Local 本地向量库。确认 backend/.env 中的 Embedding 配置可用后，重启后端并在知识库页面重新索引或新增知识片段。

## 9. 推荐演示顺序

1. 登录系统
2. 查看工作台
3. 进入数据源页面，查看或上传 CSV
4. 进入智能分析页面，输入“分析近10天各地区销售额趋势”
5. 继续追问“按产品类别拆分”
6. 查看图表、关键发现、查询结果和 SQL
7. 打开历史会话，展示多轮会话记录
8. 进入知识库页面，新增或删除知识片段
9. 进入账户管理页面，演示管理员权限
```
